# 🎫 Support Ticket System

> Веб-система регистрации и обработки заявок технической поддержки с AI-классификацией обращений через Google Gemini

## 📋 Обзор

Система автоматизирует работу с заявками техподдержки — от создания до закрытия и анализа статистики. Не нужна тяжёлая инфраструктура: всё это один Node.js-процесс с базой SQLite, запускаемый одной командой. Главная фишка — заявки анализирует AI: определяет категорию, приоритет и даёт рекомендации, а отдельный модуль прогнозирует время обработки на основе истории.

## ✨ Возможности

- 🔐 **Аутентификация** — JWT-токены, роли `user`/`admin`, у админов есть специализация по категории
- 🔄 **Жизненный цикл заявки** — новая → в работе → закрыта, с авторасчётом времени обработки
- 🤖 **AI-анализ** — Gemini определяет категорию, приоритет и рекомендации (с фоллбеком на 3 модели)
- ⏱️ **Прогноз SLA** — медианное время закрытия похожих заявок из истории
- 🔍 **Поиск и фильтры** — по тексту и статусу, оценка закрытых заявок 1–5
- 👨‍💼 **Очередь админа** — смена статуса, ручная коррекция категории/приоритета, подсветка просрочки
- 📊 **Аналитика** — диаграммы на чистом CSS, без сторонних библиотек
- 📜 **История изменений** — полный журнал всех действий с заявками
- 👥 **Управление аккаунтами** — создание и удаление пользователей и админов

## 🏗️ Архитектура

Три уровня:

- 🖥️ **Представление** — статические HTML-страницы + JS-модули, без фреймворков
- ⚙️ **Бизнес-логика** — единый `server.js` (Express): аутентификация, роли, REST-маршруты
- 🗄️ **Хранение** — файл SQLite через промис-обёртки над `sqlite3`

AI-анализ — прямой REST-вызов Gemini из сервера, без SDK.

## 📁 Структура репозитория

```text
support-system/
├── server.js                                  # Express-сервер, API, инициализация БД
├── api.js                                      # Общий клиентский слой (fetch + JWT)
├── auth.html / auth.js                         # Вход и регистрация
├── dashboard_user.html / dashboard_user.js     # Кабинет пользователя
├── dashboard_admin.html / dashboard_admin.js   # Админ-панель
├── styles.css                                  # Стили (glassmorphism)
├── database.sqlite                             # БД (создаётся автоматически)
└── package.json / package-lock.json
```

## 🚀 Запуск

```bash
npm install
node server.js
```

Открыть `http://localhost:3000`. Логин по умолчанию: `admin` / `1234`
